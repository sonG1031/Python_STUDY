문자열을 사용하다 보면 문자열 안에 작은따옴표나 큰따옴표를 넣어야 할 경우가 생긴다.
문자열 안에 '(작은따옴표)를 넣고 싶다면 문자열을 "(큰따옴표)로 묶고, # " ' "
문자열 안에 "(큰따옴표)를 넣고 싶다면 문자열을 '(작은따옴표)로 묶는다. # ' " '
여러 줄로 된 문자열(""", ''')은 작은따옴표 안에 작은따옴표와 큰따옴표를 둘 다 넣을 수 있다.

문자열 안에 ', " 등의 특수 문자를 포함하기 위해 앞에 \를 붙이는 방법을 이스케이프(escape)라고 부름.

리스트에 저장된 각 값 요소(element)라고 부른다.
리스트는 문자열, 정수, 실수, 불 등 모든 자료형을 저장할 수 있으며 자료형을 섞어서 저장해도 됨.
빈 리스트를 만들 때는
>>> a = [] # []
>>> b = list() # []

튜플은 리스트처럼 요소를 일렬로 저장하지만, 안에 저장된 요소를 변경, 추가, 삭제를 할 수 없다.(간단하게 읽기 전용 리스트)
괄호로 묶지 않고 값만 콤마로 구분해도 튜플이 됨!
보통 튜플은 요소가 절대 변경되지 않고 유지되어야 할 때 사용됨.
요소가 한 개인 튜플을 만들 때는 ( )(괄호) 안에 값 한 개를 넣고 ,(콤마)를 붙임.

[튜플은 요소를 변경, 추가, 삭제할 수도 없는데 값 한 개짜리 튜플은 왜 필요할까?]
함수(클래스)를 사용하다 보면 값이 아닌 튜플을 넣어야 할 경우가 생긴다.
이때 값은 한 개지만 튜플을 넣어야 할 때 (값, )과 같은 형식을 사용!
그냥 튜플 형태를 유지하기 위한 문법..

튜플과 리스트는 요소를 변경, 추가, 삭제할 수 있는지 없는지만 다를 뿐 기능과 형태는 같다.
따라서 튜플을 리스트로 만들거나 리스트를 튜플로 만들 수도 있다.

list와 tuple에 문자열을 넣으면 문자 리스트, 문자 튜플이 생성됩니다.
>>> list('Hello')
['H', 'e', 'l', 'l', 'o']

[언패킹(unpacking)] # 리스트 언패킹(list unpacking), 튜플 언패킹(tuple unpacking)
리스트와 튜플의 요소를 변수 여러 개에 할당하는 것.(단, 변수의 개수와 리스트(튜플)의 요소 개수는 같아야함.)
>>> x = [1, 2, 3]
>>> a, b, c = x # 리스트 언패킹(list unpacking)
>>> print(a, b, c)
1 2 3

[패킹(packing)] # 리스트 패킹(list packing), 튜플 패킹(tuple packing)
변수에 리스트 또는 튜플을 할당하는 과정. Ex) a = [1,2,3]
